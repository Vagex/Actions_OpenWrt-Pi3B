# Actions-OpenWrt
利用Actions云编译，整合OpenWrt固件编译与内核打包方法，一键生成可直接使用的固件。

# 目前包含的常用Packages 如下：
- [Passwall]
- [Opencalsh]
- [SSR-Plus]
- [Mosdns]
